# InvisESP 1.8.9

Client-side BedWars helper mod for **Minecraft 1.8.9** (Legacy Fabric).

## Features

- Player ESP (box outline)
- Particle ESP (witch spell detection)
- Invisibility alerts + HUD
- Cap alert (leather helmet colors)
- Bed / obsidian detection
- Auto tool switcher
- `/invisesp` commands

## Requirements

- Minecraft 1.8.9
- [Legacy Fabric Loader](https://legacyfabric.net/)
- [Legacy Fabric API](https://modrinth.com/mod/legacy-fabric-api)

## Install

1. Put `invisesp-1.0.2.jar` into your `.minecraft/mods` folder.
2. Launch with Legacy Fabric 1.8.9.

## Build

```bash
./gradlew build
```

Output jar: `invisesp-1.0.2.jar` (project root) and `build/libs/invisesp-1.0.2.jar`.
